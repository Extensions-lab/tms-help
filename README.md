# TMS for Logistics Service Providers Documentation

This repository contains user documentation for **TMS for Logistics Service Providers for Microsoft Dynamics 365 Business Central**.

Start with [index.md](index.md) or open the published help site: <https://tms.extensions-lab.com>.

## Authoring rules

- Write for forwarding coordinators, dispatchers, finance users, administrators, and integration users.
- Use short task-based sections: **Before you start**, **How to work**, **Fields that matter most**, **Good to know**, **Troubleshooting**, **Related**.
- Use relative links such as `forwardingorder.md` and `resources/topic/file.png`.
- Keep screenshots in `resources/<topic>/pics/` unless the topic already uses a different path.
- Keep this site focused on the LSP workflow.
- Use lowercase **shipper** only when it means the party that releases cargo on a Forwarding Order.
